**关键点：**

- 多项目管理（类似 workspace）

- 支持切换分支

- 拉取（pull）

# 四、关键技术实现（核心代码思路）

## 1️⃣ Git 操作封装

```
from git import Repo

class GitService:
    def __init__(self, repo_path):
        self.repo = Repo(repo_path)

    def pull(self):
        origin = self.repo.remotes.origin
        origin.pull()

    def push(self):
        origin = self.repo.remotes.origin
        origin.push()

    def commit(self, message):
        self.repo.git.add(all=True)
        self.repo.index.commit(message)
```

***
